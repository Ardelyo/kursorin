# Panduan Kontribusi Kursorin

Terima kasih atas minat Anda untuk berkontribusi pada Kursorin! Kami ingin membuat proyek ini bermanfaat bagi sebanyak mungkin orang, terutama mereka yang membutuhkan alat bantu aksesibilitas.

## Cara Berkontribusi

### 1. Melaporkan Bug
Jika Anda menemukan bug, silakan buat **Issue** baru di GitHub dengan menyertakan:
- Langkah-langkah untuk mereproduksi bug.
- Apa yang Anda harapkan terjadi vs apa yang sebenarnya terjadi.
- Screenshot atau log error jika ada.

### 2. Mengajukan Fitur Baru
Punya ide keren? Buat **Issue** dengan label `enhancement` dan jelaskan ide Anda secara detail. Diskusikan dulu sebelum mulai coding agar tidak sia-sia.

### 3. Pull Request (PR)
1.  **Fork** repositori ini.
2.  Buat branch fitur baru: `git checkout -b fitur-keren-anda`.
3.  Commit perubahan Anda: `git commit -m 'Menambahkan fitur keren'`.
4.  Push ke branch: `git push origin fitur-keren-anda`.
5.  Buat **Pull Request** di GitHub.

## Standar Kode
- Gunakan Bahasa Indonesia atau Inggris untuk komentar kode (konsisten).
- Pastikan kode Anda rapi dan mudah dibaca.
- Tes perubahan Anda sebelum mengajukan PR.

## Fokus Pengembangan Saat Ini
- **Optimasi Pelacakan Kepala**: Meningkatkan presisi dan mengurangi jitter.
- **Gestur Tangan Baru**: Menambahkan lebih banyak gestur yang dapat dikustomisasi.
- **Aksesibilitas**: Fitur yang memudahkan penggunaan bagi penyandang disabilitas.
